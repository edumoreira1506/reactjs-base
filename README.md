## Dependencies

```
NodeJS >= 10.17
Yarn >= 1.21
```

## Setup

Clone the project.
```console
$ git clone git@github.com:edumoreira1506/reactjs-base.git
$ cd reactjs-base
```

Copy the contents of the `.env.example` to `.env` then change with the credentials of your local environment.

```console
$ cp .env.example .env
```

Install all packages.
```console
$ yarn
```
